## 📌 Tech Stacks


- **Language** : Java (jdk-17)
- **Web application Framework** : Spring boot (3.2.0), Spring Data JPA
- **DataBase** : MySql (8.1.0)
- **Cloud/Infra** : Aws EC2, RDS, code deploy
- **web server** : Tomcat
- **Collaborative Tool** : Github
- **Version Control** : Git
